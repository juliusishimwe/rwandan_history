# Backend for Rwandan History Website

This backend provides a simple Express API for the frontend.

## Install

```bash
cd backend
npm install
```

## Run

```bash
npm start
```

## Run in development

```bash
npm run dev
```

## Available endpoints

- `GET /api/message`
- `GET /api/status`
- `GET /api/timeline`
- `GET /api/kings`
- `GET /api/kings/:id`
- `POST /api/contact`
- `GET /api/submissions`
